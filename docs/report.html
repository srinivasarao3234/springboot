Library Management System

Spring Boot Project Report — Books & Authors CRUD Application

Student Project — Spring Boot application managing two entities (Books and Authors) with Create, Read, and Update operations using JPA, JSP views, and unit testing.
GitHub URL: https://github.com/srinivasarao3234/springboot.git

1. Technology Stack

Java 21 Spring Boot 3.2.5 Spring Data JPA Hibernate H2 Database JSP + JSTL JUnit 5 Mockito Maven

2. Entity Relationship Design

The application manages two entities: Author and Book, connected by a One-to-Many relationship.

👤 Author (authors)

  • 🔑 id — BIGINT, PK
  • name — VARCHAR, NOT NULL
  • email — VARCHAR, UNIQUE
  • nationality — VARCHAR
  • bio — VARCHAR(500)
1 → *

📖 Book (books)

  • 🔑 id — BIGINT, PK
  • title — VARCHAR, NOT NULL
  • isbn — VARCHAR, UNIQUE
  • genre — VARCHAR
  • price — DOUBLE
  • publication_year — INTEGER
  • 🔗 author_id — FK → authors(id)

4. Implementation Highlights

4.1 Custom Join Query

@Query("SELECT new com.library.dto.BookAuthorDTO(" +
       "b.id, b.title, b.isbn, b.genre, b.price, b.publicationYear, " +
       "a.id, a.name, a.email, a.nationality) " +
       "FROM Book b INNER JOIN b.author a ORDER BY b.title")
List<BookAuthorDTO> findAllBooksWithAuthors();
